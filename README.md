# 【技术特性】
采用内存映射（Memory-Mapped File）技术实现高效解析：   
✓ 零拷贝数据访问 - 直读磁盘文件无需全量加载  
✓ 毫秒级响应 - 百万次查询/秒的处理性能  
✓ 低内存占用 - 常驻内存≤4MB  
✓ 多线程安全 - 支持高并发访问  

# 【数据覆盖说明】
当前版本支持：

中国大陆 IP 段精准定位（区县级）

基础运营商信息解析

ASN 网络标识识别

已知限制：
× 港澳台地区定位精度受限
× 海外 IP 仅显示国家级信息
× 部分云服务商 IP 识别偏差
